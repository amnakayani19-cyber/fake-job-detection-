# fake-job-detection-

## Run the Streamlit app

```bash
python -m streamlit run fakejobpostings_streamlit_app.py --server.headless true --server.port 8501
```

When the app starts successfully, Streamlit prints output like:

- `You can now view your Streamlit app in your browser.`
- `Network URL: http://127.0.0.1:8501`

Then open `http://127.0.0.1:8501` in your browser to interact with:
- 📊 Train Model
- 🔮 Predict Job
- 📈 Model Metrics
